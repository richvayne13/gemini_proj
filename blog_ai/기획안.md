# 블로그 자동화 시스템 구축 기획안

## 1. 개요
사용자가 입력한 키워드를 바탕으로 정보를 수집하고, AI를 활용해 콘텐츠와 이미지를 생성한 뒤, Quarto를 통해 정적 웹사이트(GitHub Pages)로 자동 게시하는 시스템을 구축한다.

## 2. 주요 기능
- **정보 수집**: Google Search API를 활용하여 특정 키워드에 대한 최신 트렌드 및 자료 수집.
- **이미지 생성**: Gemini Image Generation API(또는 관련 SDK)를 사용하여 포스트용 이미지 생성.
- **콘텐츠 생성**: 수집된 자료를 바탕으로 Gemini 모델을 활용해 블로그 초안(Markdown) 작성.
- **자동 배포**: Quarto를 사용하여 Markdown을 HTML로 렌더링하고 GitHub Pages에 푸시.

## 3. 작업 흐름 (Workflow)
1. `search.md`: 키워드 검색 결과 저장.
2. `image.jpg`: 생성된 이미지 저장.
3. `draft.md`: 생성된 블로그 초안 저장.
4. `index.html`: 최종 렌더링된 결과물.

## 4. 기대 효과
- 콘텐츠 생산 시간 단축.
- 일관된 품질의 블로그 운영 가능.
